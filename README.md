English 🇬🇧
Tracker Comparator is a minimal, single-file HTML web app to merge, clean, and format Torrent tracker lists. It runs instantly in your browser with no installations, backends, or dependencies.
✨ Features
Duplicate Removal: Merges lists and automatically removes duplicates.
Formatting: Alphabetically sorts and separates trackers with a blank line, ready for easy pasting.
.txt Support: Load your "Master" list directly from a local file.
Continuous Mode: Paste multiple lists sequentially; the "Old List" auto-updates for the next input.
Export: One-click copy to clipboard or download as a new .txt file.
Multilingual: Built-in support for Italian and English (i18n).
Responsive: Clean Tailwind CSS interface, optimized for desktop and mobile.
🚀 How to use
Usage is immediate:
Download and open tracker_updater.html in any modern web browser.
Paste existing trackers in the "Old List" (or load from file) and new trackers in the "New List".
Click "⚡ Merge, Clean and Format".
Copy the result or download the updated text file.
🛠️ Technologies Used
HTML5 & Vanilla JavaScript
Tailwind CSS (via CDN)
Native Web APIs (Set, FileReader, Blob)

Italiano 🇮🇹
Tracker Comparator è una web-app HTML in un singolo file, minimale e moderna, per unire, pulire e formattare liste di tracker Torrent. Funziona istantaneamente nel browser, senza installazioni o dipendenze.
✨ Funzionalità
Rimozione Duplicati: Unisce le liste eliminando automaticamente i doppioni.
Formattazione: Ordina alfabeticamente e separa i tracker con un rigo vuoto, pronti da incollare.
Supporto .txt: Carica la tua lista "Master" direttamente da un file locale.
Modalità Continua: Incolla liste in sequenza; la "Lista Vecchia" si aggiorna automaticamente.
Esporta: Copia negli appunti con un clic o scarica come nuovo file .txt.
Multilingua: Supporto integrato per Italiano e Inglese (i18n).
Responsive: Interfaccia pulita in Tailwind CSS, ottimizzata per desktop e mobile.
🚀 Come si usa
L'utilizzo è immediato:
Scarica e apri tracker_updater.html in qualsiasi browser moderno.
Incolla i vecchi tracker nella "Lista Vecchia" (o caricali da file) e i nuovi nella "Lista Nuova".
Clicca "⚡ Unisci, Pulisci e Formatta".
Copia il risultato o scarica il file testuale aggiornato.
🛠️ Tecnologie Utilizzate
HTML5 & Vanilla JavaScript
Tailwind CSS (tramite CDN)
API Web Native (Set, FileReader, Blob)
