🧲 Tracker Comparator

Read this in other languages: English | Italiano

English 🇬🇧

Tracker Comparator is a single-file HTML web app with a minimal and modern design, built to merge, clean, and format Torrent tracker lists.

No installations, no backend, or complex dependencies: it does everything instantly inside your browser.

✨ Features

Duplicate Removal: Merges your old lists with new trackers, automatically removing duplicates.

Perfect Formatting: Generates an alphabetically sorted list separated by a blank line, ready to be pasted into your favorite Torrent client.

.txt File Support: Load your "Master" list directly from a local text file.

Continuous Mode: Paste multiple lists in sequence. The app will automatically update the "Old List", preparing for the next input.

Copy & Download: Copy the result to your clipboard with one click or download it as a new .txt file.

Multilingual: Built-in support for Italian and English (i18n).

Responsive & Fast: Clean user interface built with Tailwind CSS, perfectly functional on desktop and mobile.

🚀 How to use

Being a "Single-File" application, the usage is immediate:

Download the tracker_updater.html file.

Double click to open it in any modern web browser (Chrome, Firefox, Edge, Safari).

Paste your existing trackers in the "Old List" field (or load them from a file).

Paste the new trackers to check in the "New List" field.

Click on "⚡ Merge, Clean and Format".

Done! Copy the result or download the updated text file.

🛠️ Technologies Used

HTML5 & Vanilla JavaScript (No heavy JS frameworks)

Tailwind CSS (via CDN for minimal and responsive styling)

Native Web APIs (Set for duplicate removal, FileReader for upload, Blob for download)

Italiano 🇮🇹

Tracker Comparator è una web-app in un singolo file HTML, dal design minimale e moderno, progettata per unire, pulire e formattare liste di tracker Torrent.

Niente installazioni, niente backend o dipendenze complesse: fa tutto istantaneamente all'interno del tuo browser.

✨ Funzionalità

Rimozione dei Duplicati: Unisce le tue vecchie liste con i nuovi tracker eliminando automaticamente i doppioni.

Formattazione Perfetta: Genera una lista ordinata alfabeticamente e separata da un rigo vuoto, pronta per essere incollata nel tuo client Torrent preferito.

Supporto File .txt: Carica la tua lista "Master" direttamente da un file testuale locale.

Modalità Continua: Incolla liste multiple in sequenza. L'app aggiornerà automaticamente la "Lista Vecchia" preparandosi per la prossima iniezione.

Copia & Scarica: Copia il risultato negli appunti con un clic o scaricalo come nuovo file .txt.

Multilingua: Supporto integrato per Italiano e Inglese (i18n).

Responsive & Veloce: Interfaccia utente pulita costruita con Tailwind CSS, perfettamente funzionante su desktop e mobile.

🚀 Come si usa

Essendo un'applicazione "Single-File", l'utilizzo è immediato:

Scarica il file tracker_updater.html.

Fai doppio clic per aprirlo in qualsiasi browser web moderno (Chrome, Firefox, Edge, Safari).

Incolla i tuoi tracker già in possesso nel campo "Lista Vecchia" (o caricali da file).

Incolla i nuovi tracker da verificare nel campo "Lista Nuova".

Clicca su "⚡ Unisci, Pulisci e Formatta".

Fatto! Copia il risultato o scarica il file testuale aggiornato.

HTML5 & Vanilla JavaScript (Nessun framework JS pesante)

Tailwind CSS (tramite CDN per lo styling minimale e responsive)
